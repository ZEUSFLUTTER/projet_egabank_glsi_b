\# TP JEE – Application Banque



\## Informations générales

\- \*\*Nom du projet\*\* : Banque-ega

\- \*\*Matière\*\* : JEE

\- \*\*Année universitaire\*\* : 2025–2026

\- \*\*Étudiant\*\* : BILLA Madjide

\- \*\*Classe\*\* : GLSIA



\## Description

Ce projet est une application bancaire développée dans le cadre d’un TP JEE.

Elle permet la gestion des comptes et des opérations bancaires via une architecture

séparée frontend / backend.



\## Architecture

Le projet est divisé en deux parties :



\- \*\*backend/\*\* : application Java (JEE / Spring Boot)

\- \*\*frontend/\*\* : application Angular



\## Technologies utilisées

\### Backend

\- Java

\- JEE / Spring Boot

\- Maven

\- Base de données : MySQL 



\### Frontend

\- Angular

\- TypeScript

\- HTML / CSS
